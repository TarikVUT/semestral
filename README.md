#For Fedora
1. install gnome-shell-extension
```bash
sudo dnf install gnome-shell-extension* -y
```
2. command to disable the automatic Screen Lock in Fedora Workstation
```bash
dconf write /org/gnome/desktop/screensaver/lock-enabled false
```
3. reload gnome
```bash
killall -u user
```
